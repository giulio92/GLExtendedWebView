## What it is
GLExtendedWebView is a drop-in, ready to use, `WKWebView` subclass that brings Interface Builder compatibility to `WKWebView`, something that Apple forgot.

## Requirements
- Xcode 8.0+
- Swift 3.0+
- iOS 8.0+
